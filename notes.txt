2. apt-get install git vim redshift python3-dev python-pip python3-pip build-essential virtualenvwrapper

3. apt-get purge pidgeon thunderbird brasero

4. change firefoxs in about:config - layout.css.devPixelsPerPx

5. Add udev rules for android usb devices -> $ wget -S -O - http://source.android.com/source/51-android.rules | sed "s/<username>/$USER/" | sudo tee >/dev/null /etc/udev/rules.d/51-android.rules; sudo udevadm control --reload-rules

6. Set up firefox settings to minimize disk writes

   - about:config
   - browser.sessionstore.interval. Default set to 15 seconds change to 30m
